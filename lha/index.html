LHa for UNIX RPM

English Page

このページでは、 OSDN で公開されている LHa for UNIX autoconf 版の RPM を公開しています。

LHa for UNIX autoconf 版は、岡本継男さんによって公開されている LHa for UNIX に新井康司さんによる autoconf 化パッチを当てたものになります。

注意事項

本ページにて現在公開しているバージョンには脆弱性が存在します。GitHub リポジトリにある最新のソースコードを利用してください。

RPM の作成環境は Vine Linux 3.2 (glibc 2.3.3 , gcc 3.3.2) です。

Vine Linux 4.0 以降の lha は autoconf 版となっておりますので、本ページで公開している RPM を使用する必要はありません。 lha コマンド実行時に --system-kanji-code= オプションでシステムの文字コードを指定してください。

binary RPM はシステムの文字コードを EUC-JP としてビルドしたものになります。他の文字コードの場合は lha コマンド実行時に --system-kanji-code= オプションで指定するか、デフォルト設定を変更する場合は configure オプション --enable-multibyte-filename= でシステムの文字コードを指定して source RPM からビルドし直す必要があります。また、 spec ファイルの文字コードも EUC-JP となっております。
