LHa for UNIX RPM (old versions)

Old versions of LHa for UNIX RPM.

Notice

The old versions published on this page are unsupported and vulnerable. Use the latest source code on GitHub repository.

These RPMs are made on Vine Linux 2.X - 3.2.

Vine Linux 4.0 and later includes autoconfiscated version of LHa, so you don't need to use RPMs distributed here. Use with "--system-kanji-code=" option.

Binary RPMs are built for EUC-JP encoding. If you use other system encodings, you should use lha command with "--system-kanji-code=" option, or rebuild from source RPM with "--enable-multibyte-filename=" configure option.

Spec file is also written in EUC-JP encoding.

Cygwin version is built for Shift_JIS encoding.
