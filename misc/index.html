isodatetime.cmd

Windows のコマンドプロンプト上で、 ISO8601 形式の日付・時刻を取得するためのコマンドファイルです。

isodatetime.cmd をダウンロードし、 PATH の通っている場所に置いてください。

+09:00 以外のタイムゾーンには対応しておりません。他のタイムゾーンで使用する場合は適切に編集する必要があります。version 1.5 でタイムゾーンの出力は削除しました。

isodatetime.cmd version 1.5

実行例

C:\>isodatetime /h
usage : isodatetime [options]
options : /h, /? : show this help
          /e     : extended format
          /f     : add fractional part of a second

C:\>isodatetime
20100311T203521

C:\>isodatetime /e
2010-03-11T20:35:29

C:\>isodatetime /f
20100311T203654,34

C:\>isodatetime /e /f
2010-03-11T20:37:32,54
